# 49 - Open Source Licenses

**Project:** Ravenshaw Moments\
**Version:** 2.0 (Legacy)\
**Status:** Production Ready\
**Document Type:** Open Source Licenses & Attribution

**Effective Date:** To be announced

------------------------------------------------------------------------

# 1. Purpose

This document records the open-source software used by Ravenshaw Moments
and describes the project's commitment to complying with applicable
open-source licenses.

------------------------------------------------------------------------

# 2. Commitment

Ravenshaw Moments values the open-source community and aims to:

-   Respect license terms
-   Preserve required copyright notices
-   Provide attribution where required
-   Monitor dependency licenses
-   Replace incompatible dependencies when necessary

------------------------------------------------------------------------

# 3. Core Technologies

The project currently uses (or is designed to use) technologies
including:

-   Next.js
-   React
-   TypeScript
-   Tailwind CSS
-   Supabase
-   PostgreSQL
-   Shadcn UI
-   Lucide Icons
-   React Hook Form
-   Zod
-   Sonner
-   date-fns
-   class-variance-authority
-   clsx
-   next-themes

Each dependency remains subject to its own license.

------------------------------------------------------------------------

# 4. License Compliance

The Platform will:

-   Retain required license notices
-   Include attribution where required
-   Respect copyright notices
-   Avoid unauthorized redistribution of proprietary software

------------------------------------------------------------------------

# 5. Dependency Management

Before introducing a new dependency:

-   Review the license
-   Verify commercial compatibility
-   Evaluate maintenance status
-   Assess security risks

------------------------------------------------------------------------

# 6. Third-Party Assets

Images, fonts, icons, datasets, and media must be used only when
permitted by their respective licenses.

Where attribution is required, it should be provided appropriately.

------------------------------------------------------------------------

# 7. Source Code

Unless separately stated, Ravenshaw Moments source code remains the
intellectual property of its owners.

Publication of source code does not automatically grant permission for
unrestricted commercial use.

------------------------------------------------------------------------

# 8. Future Reviews

Periodic dependency reviews should include:

-   License changes
-   Security advisories
-   End-of-life packages
-   Replacement planning

------------------------------------------------------------------------

# 9. Disclaimer

This document provides a high-level overview and does not replace the
individual license text supplied by each dependency.
