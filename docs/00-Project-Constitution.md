**Project:** Ravenshaw Moments\
**Version:** 2.0 (Legacy)\
**Status:** 🔒 Frozen\

**Priority:** Highest (Read First)

------------------------------------------------------------------------

# Purpose

This document is the supreme governing document for the Ravenshaw
Moments project.

Every developer, AI coding assistant, contributor, maintainer, designer,
and administrator must follow this constitution before making any
architectural, technical, business, or design decision.

If any implementation conflicts with this constitution, **the
constitution takes precedence**.

------------------------------------------------------------------------

# Vision

To build the most comprehensive, secure, privacy-focused,
community-driven digital ecosystem for the Ravenshaw University
community while preserving its memories, achievements, history, and
legacy for future generations.

------------------------------------------------------------------------

# Mission

Ravenshaw Moments exists to:

-   Preserve student memories.
-   Celebrate achievements.
-   Connect students, teachers, alumni, organizations, and supporters.
-   Digitally archive Ravenshaw University's heritage.
-   Build a trustworthy, secure, and scalable platform.

------------------------------------------------------------------------

# Core Values

## Student First

Every feature must ultimately benefit the community.

## Privacy by Default

Only the minimum information required is public. Users control their
visibility wherever practical.

## Security First

Security is never sacrificed for convenience.

## Community Driven

Students, teachers, alumni, organizations, and supporters are equal
members of the ecosystem.

## Transparency

Moderation, certificates, contributions, and recognition should be fair
and auditable.

## Accessibility

Design for everyone regardless of device or ability.

## Scalability

Architecture should support long-term growth without major redesign.

## Maintainability

Prefer modular, documented, and readable solutions.

## Authenticity

Preserve genuine Ravenshaw history and memories without rewriting or
fabricating records.

## Independence

Ravenshaw Moments is an independent community platform unless a formal
partnership is established.

------------------------------------------------------------------------

# Documentation First

No feature may be implemented before it is documented.

The `/docs` directory is the single source of truth.

------------------------------------------------------------------------

# Architecture Rules

-   Preserve modular architecture.
-   Keep backward compatibility whenever possible.
-   Database schema changes require documentation updates.
-   Security-sensitive changes require review.

------------------------------------------------------------------------

# Data Governance

-   Collect only necessary data.
-   Protect personal information.
-   Respect user privacy choices.
-   Participation certificates remain private.
-   Winner certificates are public unless policy specifies otherwise.

------------------------------------------------------------------------

# Legal Principles

-   Respect copyright.
-   Respect privacy.
-   Respect applicable laws.
-   Clearly identify the platform as independent from Ravenshaw
    University.
-   Encourage users to verify official information from official
    university sources.

------------------------------------------------------------------------

# AI Development Rules

Every AI assistant must:

1.  Read the documentation before coding.
2.  Explain significant changes.
3.  Avoid destructive edits.
4.  Preserve authentication and security.
5.  Never weaken Row Level Security (RLS).
6.  Follow coding standards and project architecture.

------------------------------------------------------------------------

# Governance

Version 2.0 is frozen.

Future enhancements must: - be documented first, - be reviewed, - then
be implemented.

------------------------------------------------------------------------

# Approval

This constitution governs Ravenshaw Moments Version 2.0 and remains
effective until superseded by a future major version.
