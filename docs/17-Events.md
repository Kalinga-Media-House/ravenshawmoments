**Project:** Ravenshaw Moments\

**Document Type:** Event Management Specification

# Purpose

This document defines the complete event management lifecycle for
Ravenshaw Moments, including event creation, registration, attendance,
media management, notifications, and archival.

------------------------------------------------------------------------

# Objectives

-   Organize and showcase university events.
-   Simplify event registration and attendance.
-   Preserve event memories through media archives.
-   Provide transparent event management.

------------------------------------------------------------------------

# Event Information

Every event should include:

-   Title
-   Description
-   Banner Image
-   Date
-   Start Time
-   End Time
-   Venue / Location
-   Organizer
-   Department / Hostel / Organization
-   Registration Deadline
-   Capacity (optional)
-   Contact Person
-   Event Status

------------------------------------------------------------------------

# Registration

Supports:

-   Open Registration
-   Approval-based Registration
-   Invitation-only Events

Participants may register as:

-   Student
-   Teacher
-   Alumni
-   External Guest

------------------------------------------------------------------------

# Attendance

Supported methods:

-   QR Code Check-in
-   Manual Attendance
-   Admin Verification

Attendance records are stored for reporting and certificate eligibility.

------------------------------------------------------------------------

# Speakers & Guests

Events may include:

-   Chief Guest
-   Guest of Honour
-   Speakers
-   Judges
-   Faculty Coordinators
-   Organizing Committee

------------------------------------------------------------------------

# Media System

After an event:

-   Upload Photos
-   Upload Videos
-   Create Event Album
-   Feature Selected Memories

Media becomes part of the permanent university archive.

------------------------------------------------------------------------

# Notifications

Participants receive:

-   Registration Confirmation
-   Event Reminder
-   Venue Changes
-   Schedule Updates
-   Event Cancellation
-   Thank You Message (optional)

------------------------------------------------------------------------

# Certificates

Where applicable:

-   Participation Certificates
-   Volunteer Certificates
-   Organizer Certificates
-   Appreciation Certificates

Certificate generation follows the Certificate Policy.

------------------------------------------------------------------------

# Event Archive

Every completed event retains:

-   Event Details
-   Gallery
-   Participant Count
-   Organizer Information
-   Achievements
-   Related Certificates

Archived events remain searchable.
