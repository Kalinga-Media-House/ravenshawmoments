# About

Ravenshaw Moments is an independent, community-driven digital ecosystem
designed for the Ravenshaw University community. It preserves memories,
celebrates achievements, connects students, teachers, alumni,
organizations, and supporters, and builds a permanent digital heritage
archive.

This repository follows a **Documentation First** approach.

The `/docs` directory is the **single source of truth** for every
architectural, functional, technical, security, legal, and design
decision.
