# 21 - Hostels

**Project:** Ravenshaw Moments\
**Version:** 2.0 (Legacy)\
**Status:** ✅ Complete\
**Document Type:** Hostel & Housing Hub Specification

------------------------------------------------------------------------

# Purpose

This document defines the official hostel ecosystem for Ravenshaw
Moments, covering university hostels, Hostel BMC administration, hostel
communities, and the Housing Hub for sponsored private hostels.

------------------------------------------------------------------------

# Objectives

-   Create a digital home for every hostel.
-   Preserve hostel memories and history.
-   Improve communication among residents.
-   Help new students discover accommodation.
-   Provide a transparent sponsored hostel directory.

# Housing Hub

A **Private Hostels Near Me** button appears prominently on the hostel
landing page.

Purpose:

Help students discover verified and sponsored private accommodation near
Ravenshaw University.

------------------------------------------------------------------------

# Private Hostel Listings

Every listing includes:

-   Hostel Name
-   Cover Photo
-   Gallery
-   Address
-   Google Maps Location
-   Contact Number
-   Owner Name
-   Facilities
-   Rent Information (optional)
-   Room Types
-   Availability Status

------------------------------------------------------------------------

# Sponsored Listings

Super Admin may:

-   Approve listings
-   Feature sponsored hostels
-   Verify information
-   Remove expired listings

Sponsored listings receive higher visibility.

------------------------------------------------------------------------

# Student Inquiry

Students may:

-   View hostel details
-   Contact owner
-   Submit inquiry
-   Save favourite hostels (future)
